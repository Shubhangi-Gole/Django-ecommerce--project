# Django E-Commerce Project

## Features

* User Registration
* User Login & Logout
* Product Listing
* Product Search
* Product Detail Page
* Add to Cart
* Remove from Cart
* Checkout Page
* Order Success Page

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap

## How to Run

1. Clone the repository
2. Create a virtual environment
3. Install requirements

```bash
pip install -r requirements.txt
```

4. Run migrations

```bash
python manage.py migrate
```

5. Start server

```bash
python manage.py runserver
```
